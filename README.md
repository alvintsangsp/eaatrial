# 香港地產代理人員資格考試（EAQE）/ 營業員資格 (SQE) 模擬考試應用程式

免費的香港地產代理人員資格考試（EAQE）/ 營業員資格 (SQE) 練習平台，幫助您準備地產代理資格考試。包含模擬考試和練習模式。

## 功能特點

- 📝 **模擬考試模式**：30題隨機選擇的題目，60分鐘計時考試
- 📚 **練習模式**：按主題分類練習，每個主題20題
- ⏱️ **計時功能**：考試倒數計時器
- 📊 **成績分析**：詳細的答題結果和解析
- 📱 **響應式設計**：支援手機、平板和桌面設備
- 🌐 **雙語支援**：繁體中文和簡體中文切換

## 技術棧

- React + TypeScript
- Vite
- shadcn/ui 組件庫
- Tailwind CSS
- React Router
- Zustand 狀態管理

## 開發環境

### 安裝依賴

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

### 構建生產版本

```bash
npm run build
```

### 預覽生產版本

```bash
npm run preview
```
